### Cloud Server Configuration

---

Проект, имитирующий интерфейс заказа облачного сервиса.
Релизован калькулятор стоимости, валидация конфигурации и адаптивный UI.

**Try it:** [// пока без ссылки]

---

### Stack

- **Core:** React, TypeScript, Vite
- **State Management:** Zustand
- **UI:** Tailwind CSS, Shadcn/ui

### Функционал

- **Калькулятор:** Цена пересчитывается Real-time.
- **Выбор типа диска:** SSD / NVMe.
- **Платные опции:** Учет стоимости лицензии Windows и бэкапов.
- **Адаптив:** Полная поддержка мобильных устройств.
